# Menu Version
* Note: the menu version is not a separate version. It is a version with a customizable interface that can be adjusted directly, unlike the manual workflow used by the loader.
* You can create your own menu or optimize it based on your personal needs or working environment.
* Menu versions typically include better configuration features such as save/load, depending on the creator’s focus.
* You can refer to the documentation here: [Docs](https://github.com/K-M19/FurTDS/tree/main/Docs).
