# How to upload file Rec

After you finish recording with **Rec**, upload the **raw file** to **[github.com](https://github.com/)** or **[pastebin.pl](https://pastebin.pl/)** or **[pastefy.app](https://pastefy.app/)**.
However, **do not use pastebin.com** if you want to share it with everyone, because pastebin.com is extremely strict about **IP bans on certain countries or territories**, including those that do not support Ukraine—even if they only submitted a blank vote.
